# LittleLemonCapstone

Android Jetpack Compose app for the Little Lemon capstone project.

## How to run

1. Open this project in **Android Studio** (tested with Android Studio 2025.1.3 Narwhal).
2. Let Gradle sync finish.
3. Run the app on an emulator or physical device (minSdk 24).
4. First screen: Onboarding (register name + email).
5. Then: Home screen with hero, search, menu list, and Profile screen via header avatar.
